# 🎬 Movie Recommendation System

## 📌 Description

This project is a simple **Movie Recommendation System** developed using **Python** and **Content-Based Filtering**.

The system recommends similar movies based on their genres using **CountVectorizer** and **Cosine Similarity** from Scikit-Learn.

This project was created as part of the **CodSoft Artificial Intelligence Internship**.

---

## 🚀 Features

- Recommend similar movies
- Content-Based Filtering
- Genre-based recommendations
- Simple Command Line Interface
- Fast and Beginner Friendly

---

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-Learn
- NumPy

## 🧠 Recommendation Technique

This project uses **Content-Based Filtering**.

It converts movie genres into numerical vectors using **CountVectorizer** and calculates movie similarity using **Cosine Similarity**.
